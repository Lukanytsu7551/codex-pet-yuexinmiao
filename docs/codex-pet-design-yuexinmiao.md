# Codex 专属桌宠设计稿：月薪喵皮肤

## 设计定位

**名字**：月薪喵 for Codex  
**一句话**：一只打工感很强的表情包小猫，帮你盯 Codex 的工作、等待、完成和出错状态。  
**使用范围**：建议仅用于个人本机 Codex，不建议公开分发。LLMPET 的月薪喵素材注明来源为抖音博主 @月薪喵，公开发布或商用前需要单独确认授权。

这个方案选择**全部采用 LLMPET 仓库里的月薪喵 GIF 图片**，不再重新生成相似风格。核心气质保留为：打工、摸鱼、等回复、崩溃、收工。Codex 原生 pet 的状态更少，所以需要把月薪喵的细状态压缩进 Codex v2 pet 的标准动作行。

## 视觉方向

不要把月薪喵做成“精致吉祥物”，它的魅力就是表情包式的真实打工感：

- 形象：白色或浅灰小猫，圆头、短身体、表情夸张。
- 道具：小电脑、手机、耳机、键盘、被窝、确认牌。
- 情绪：认真干活和偷偷摸鱼并存。
- 风格：GIF 表情包感、粗线条、动作夸张，小尺寸也要一眼看懂。
- 颜色：浅色猫身 + 少量蓝/绿/黄道具点缀，避免一整套都灰白无焦点。
- 禁止：直接出现可读品牌 Logo、Claude/Codex/OpenAI 字样、长文本、复杂场景。

## Codex 状态映射

| Codex 状态 | 月薪喵表现 | 对应 LLMPET 素材语义 |
|---|---|---|
| Running | 戴耳机敲键盘、对着电脑狂输出、边吃边敲 | `cat-working*`、`cat-thinking*`、`cat-talking`、`cat-juggling` |
| Needs input | 冒汗抱手机、头顶问号、举确认牌 | `cat-waiting`、`cat-needsinput` |
| Ready | 被夸夸、起身看消息、完成后短暂开心 | `cat-happy`、`cat-attention`、`cat-greet` |
| Blocked | 抱头崩溃、大哭、出错瘫倒 | `cat-error`、`cat-sad` |

## Codex V2 Sprite 行设计

Codex v2 pet 是 8×11 atlas，每格 192×208，最终图 1536×2288。月薪喵版建议这样做：

| 行 | Codex 动作 | 月薪喵设计 |
|---|---|---|
| 0 idle | 待命 | 坐在转椅上玩手机，轻微晃动 |
| 1 running-right | 拖动向右 | 被拖着滑走，手机还没放下 |
| 2 running-left | 拖动向左 | 向左滑走，尾巴/耳朵跟随 |
| 3 waving | 打招呼 | 被闹钟叫醒，抬爪打招呼 |
| 4 jumping | 完成 | 被夸后小跳一下，开心但不太浮夸 |
| 5 failed | 出错 | 抱头崩溃或趴桌，明显失败态 |
| 6 waiting | 等你 | 冒汗举牌，直视用户 |
| 7 running | 干活 | 戴耳机敲键盘或拍键盘 |
| 8 review | 审阅/思考 | 对着电脑挠头，眼睛左右扫 |
| 9 look directions 0-157.5 | 看向 8 个方向 | 主要靠眼睛、耳朵和头部转向 |
| 10 look directions 180-337.5 | 看向另外 8 个方向 | 保持同一猫脸比例，不要变成另一只猫 |

## 状态语气

月薪喵的好玩点是“它懂打工”：

- Codex 在运行：它也在上班，认真但带点社畜感。
- Codex 要你确认：它冒汗看着你，像在说“老板这个要你点一下”。
- Codex 完成：它短暂被夸夸，然后继续坐回工位。
- Codex 出错：它崩溃，但不是吓人，是“我也没想到会这样”的表情包感。

## 素材使用策略

当前采用路线：

| 路线 | 做法 | 适合情况 |
|---|---|---|
| 个人本机复用 | 直接转换 LLMPET 的 `assets/cat/*.gif`，只在你自己的 Codex 里用 | 当前选择 |
| 重新生成相似气质 | 不直接复制原 GIF，只保留“打工猫”语义重新生成 | 暂不采用 |

由于你明确要求“全部采用月薪喵相关图片”，后续实现不再调用图像生成来重画角色。转换方式应是：抽取 GIF 帧、缩放到 Codex cell、居中透明化、按 Codex 行规则拼成 atlas。

需要接受一个限制：Codex v2 的第 9-10 行是 16 个看向方向，而 LLMPET 的月薪喵 GIF 没有专门的 16 方向素材。如果坚持 100% 使用仓库图片，这两行只能用现有 GIF 帧做近似填充，不能像重新生成的宠物那样自然转头。
